# 🌸 She Can Foundation – Contact & Response Management System

A modern Flask-based web application for managing user contact messages, admin replies, and response tracking.

This project allows users to:

✅ Submit contact/support forms  
✅ Track admin replies  
✅ View response status  
✅ Admin manage messages  
✅ Reply/Delete user messages  

Built using:

- Python Flask
- MySQL
- PyMySQL
- HTML/CSS
- Font Awesome

---

# 📌 Features

## 👩 User Features

- Beautiful responsive contact form
- Submit support/help/suggestion requests
- View admin replies
- Track message status:
  - Pending
  - Replied

---

## 🛡️ Admin Features

- Admin Dashboard
- View all messages
- Reply to users
- Delete messages
- Status management
- Live message count

---
